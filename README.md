Mind Root es un proyecto diseñado para ayudarte a organizar tus recordatorios de manera simple. Su interfaz minimalista combina un menú lateral con categorías como Ideas, Tareas Diarias, Proyectos y Recordatorios, permitiendo que cada nota esté siempre al alcance de un click.

La misma incluye:

-Diseño limpio y moderno.

-Funciones basicas (agregar, eliminar y modificar).

-Gestión de categorías que facilita separar y organizar la información.

El objetivo de Mind Root es ofrecer una experiencia sencilla pero efectiva, buscamos que crear un recordatorio nos tome el menor tiempo posible para que podamos regresar a nuestra tareas.
